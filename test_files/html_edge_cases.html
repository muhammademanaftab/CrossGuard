HTML Feature Documentation

The video element is great for embedding media content.

You can use canvas for drawing graphics with JavaScript.

The dialog element creates modal popups.

Use details and summary for expandable content.

The template element stores reusable HTML fragments.

Input type="date" creates a native date picker.

Input type="email" validates email addresses.

The loading="lazy" attribute delays image loading.

Use contenteditable="true" for in-place editing.

The download attribute triggers file downloads.

Script type="module" enables ES6 modules.

Link rel="preload" preloads critical resources.

## Naming convention

Files are named `<scope>_<topic>_<author_or_qualifier>.<ext>` in lower snake_case. Prefix tells you at a glance whose document it is: `crossguard_*` = your own, `sample_thesis_*` = exemplars, `elte_*` = institutional.

Video MIME Types

Video formats tested:

  • type="video/webm" - WebM format
  • type="video/webm; codecs=vp9" - WebM with VP9
  • type="video/mp4" - MP4/MPEG-4 format
  • type="video/ogg" - OGG video format

Audio MIME Types

Audio formats tested:

  • type="audio/mpeg" - MP3 format
  • type="audio/ogg" - OGG Vorbis format
  • type="audio/wav" - WAV format
  • type="audio/webm" - WebM audio
  • type="audio/aac" - AAC format

Rel Attribute Values Test

This page tests various rel attribute values for Cross Guard validation.

Resource Hints (in head)

The following resource hints are used in this page (see page source):

dns-prefetch

  • rel="dns-prefetch" - fonts.googleapis.com
  • rel="dns-prefetch" - cdn.example.com
  • rel="dns-prefetch" - api.example.com

preconnect

  • rel="preconnect" - fonts.gstatic.com
  • rel="preconnect" - cdn.example.com (with crossorigin)
  • rel="preconnect" - api.example.com (with crossorigin="anonymous")

preload

  • rel="preload" as="style" - critical.css
  • rel="preload" as="image" - hero-image.webp
  • rel="preload" as="font" - main-font.woff2
  • rel="preload" as="script" - app.js
  • rel="preload" as="fetch" - data.json

prefetch

  • rel="prefetch" - next-page.html
  • rel="prefetch" - gallery-images.js
  • rel="prefetch" - secondary-styles.css

modulepreload

  • rel="modulepreload" - main-module.mjs
  • rel="modulepreload" - components/header.mjs
  • rel="modulepreload" - utils/helpers.mjs

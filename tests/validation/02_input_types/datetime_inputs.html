HTML5 Date/Time Input Types Test

This file tests detection of HTML5 date and time input types.

Can I Use ID: input-datetime

URL: caniuse.com/input-datetime

Feature: input type="date"

The date input type provides a date picker interface for selecting dates.

Feature: input type="time"

The time input type provides a time picker interface without timezone.

Feature: input type="datetime-local"

The datetime-local input type combines date and time selection without timezone.

Feature: input type="month"

The month input type allows selection of a month and year.

Feature: input type="week"

The week input type allows selection of a week number and year.

Referrer Policy Values Test

This page tests all referrer policy attribute values for Cross Guard validation.

Links with referrerpolicy Attribute

no-referrer

Sends no referrer information.

Link with no-referrer

no-referrer-when-downgrade

Sends full URL for same-security requests, nothing for downgrades (HTTPS to HTTP).

Link with no-referrer-when-downgrade

origin

Sends only the origin (scheme, host, port).

Link with origin

origin-when-cross-origin

Sends full URL for same-origin, only origin for cross-origin.

Link with origin-when-cross-origin

same-origin

Sends referrer for same-origin requests only.

Link with same-origin

strict-origin

Sends only origin for same-security, nothing for downgrades.

Link with strict-origin

strict-origin-when-cross-origin

Full URL for same-origin, origin for cross-origin same-security, nothing for downgrades.

Link with strict-origin-when-cross-origin

unsafe-url

Always sends full URL (not recommended for sensitive pages).

Link with unsafe-url

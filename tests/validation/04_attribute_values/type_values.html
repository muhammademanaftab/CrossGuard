Feature: type="module" (ES6 Modules)

Can I Use ID: es6-module

URL: caniuse.com/es6-module

<!-- ES6 Module -->
<script type="module">
  import { helper } from './utils.js';
  export const version = '1.0';
</script>

<!-- External module -->
<script type="module" src="app.js"></script>

<!-- Fallback for non-module browsers -->
<script nomodule src="legacy.js"></script>

ES6 modules enable native JavaScript module support in browsers.

ES Module Import/Export Example

<script type="module">
  // Named imports
  import { add, subtract } from './math.js';

  // Default import
  import Calculator from './calculator.js';

  // Dynamic import
  const module = await import('./dynamic.js');

  // Re-export
  export { add } from './math.js';
</script>
